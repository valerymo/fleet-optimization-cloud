# fleet-optimization-cloud
fleet-optimization

This project is in development.
The project is based on legacy FleetOptimizer project.
It will have Cloud, Microservices based Architecture, multiplatform, and will work in Kubernetes on any of available Cloud platforms as Azure, AWS, GCP.

ShipDesign module developed using ideas from following Fundamental ShipDesign Theory works:
1. Ashik V.V. Proektirovanie sudov. Leningrad: Shipbuilding, 1985
2. Nogid L.M. Proektirovanie morskih sudov. Leningrad: Shipbuilding, 1976
